## Fields Reference

| Field       | Required    | Description                                      |
|-------------|-------------|--------------------------------------------------|
| Date        | Yes         | Format: `M-D-YYYY` (e.g., `1-3-2025`)            |
| Title       | No          | Overrides the filename-based title               |
| Description | Recommended | Meta description for SEO & social sharing        |
| Tags        | No          | Comma-separated, lowercase                       |
| Image       | No          | Cover image for social sharing (og:image)        |
| Draft       | No          | Set `true` to exclude from build                 |

## Filename Convention

Format: `[Number]. [Title].md`

Examples:
- `1. Eval.md` → title "Eval", slug "1-eval"
- `2. My First Week at Bootcamp.md` → title "My First Week at Bootcamp"
- `3. LeetCode Grind.md` → title "LeetCode Grind"

The number prefix controls the order and becomes part of the URL slug.
